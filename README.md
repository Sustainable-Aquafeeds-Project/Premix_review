# Premix_review

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20496250.svg)](https://doi.org/10.5281/zenodo.20496250)

This is the repository for Penrose et al 'Uncharted links from mining to fed animal systems'. This repository supports a review of how mineral supplements are represented in life cycle assessments of farmed animal feeds, highlighting their frequent exclusion or misrepresentation and the need for better integration in sustainability assessments.


Please read this file before trying to reproduce the output from this research project. Below you will find information on the publication associated with this repository and contact information for the lead author.

## Reproducibility
The code in this repository enables a user to recreate the results outlined in the above publication. 

- The code must be run as an R project - the code within relies on relative file paths from the project home folder.
- Once the R project is cloned and data folders are in place, figures and plot can be replicated.
